# Elevator System

Simple Java implementation of an elevator scheduler.

## Structure

```
elevator/
├── src/com/example/elevator/
│   ├── Direction.java
│   ├── ElevatorStatus.java
│   ├── Request.java
│   ├── Door.java
│   ├── Elevator.java
│   ├── ElevatorController.java
│   └── ElevatorSystem.java
├── compile.bat
├── compile.sh
└── README.md
```

## Run

Windows:

```bash
.\compile.bat
java -cp bin com.example.elevator.ElevatorSystem
```

Linux/Mac:

```bash
bash compile.sh
java -cp bin com.example.elevator.ElevatorSystem
```
